Os patches são atualizações para seu jogo. A equipe Spore lançava, na época de lançamento do jogo, certa regularidade pacotes que traziam correções, melhorias e novas funcionalidades para o jogo. É extremamente recomendável que você tenha todos os patches para um jogo mais estável e seguro.

Nota: caso você tenha instalado um pacote de partes ou expansão após a instalação de um patch, será necessário reinstalar a atualização.

Os patches listados nessa página são arquivos executáveis para sistemas Windows. Para jogadores em Mac OS X, basta iniciar o jogo para fazer o download da última atualização.

Patches aplicáveis para:

  • Spore: Criador de Criaturas — versão de avaliação (PC/Mac);
  • Spore: Criador de Criaturas — versão completa (PC/Mac).
