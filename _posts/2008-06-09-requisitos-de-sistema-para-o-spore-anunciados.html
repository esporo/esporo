A Electronic Arts e a Maxis anunciaram hoje os requerimentos de sistema para executar o Spore. Segue abaixo, tanto para Windows quanto para Mac.

Para Windows XP:

  • Processador Pentium 4 de 2GHz ou equivalente;
  • 512 MB de memória RAM;
  • Placa de vídeo com 128 MB de memória dedicada e suporte a Pixel Shader 2.0;
  • 6GB de espaço livre no disco rígido;

Para Windows Vista:

  • Processador Pentium 4 2GHz ou equivalente;
  • 768 MB de memória RAM;
  • Placa de vídeo com 128 MB de memória dedicada e suporte a Pixel Shader 2.0;
  • 6GB de espaço livre no disco rígido.

Para computadores com placas de vídeo não-dedicadas:

  • Processador Pentium D 2.6GHz ou Intel Core 2 Duo 1.8GHz ou equivalente;
  • 768 MB de memória RAM;
  • Placa de vídeo Intel Integrated Chipset 945GM ou superior.

Placas de vídeo suportadas:

  • Família NVIDIA GeForce: FX5900, FX5950, 6200, 6500, 6600, 6800, 7200, 7300, 7600, 7800, 7900, 7950, 8400, 8500, 8600, 8800;
  • Família ATI Radeon: 9500, 9600, 9800, X300, X600, X700, X800, X850, X1300, X1600, X1800, X1900, X1950, 2400, 2600, 2900, 3650, 3850;
  • Intel Extreme Graphics: GMA 950, GMA X3000, GMA X3100.

Para Mac OS:

  • Mac OS X 10.5.3 — Leopard ou superior;
  • Processador Intel Core Duo;
  • 1GB de memória RAM;
  • ATI Radeon X1600 ou NVIDIA GeForce 7300 GT com 128MB de memória dedicada e suporte a Pixel Shader 2.0, ou Intel Integrated GMA X3100;
  • No mínimo 5GB de espaço livre no disco rígido, mais espaço adicional para armazenamento de criações.
